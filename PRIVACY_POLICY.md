# Privacy Policy for GitHub Zread Button

**Last Updated**: Nov 10, 2025

## Overview

GitHub Zread Button is a browser extension that adds a "Zread" button to GitHub repository pages for quick access to documentation on Zread.ai.

## Data Collection

**GitHub Zread Button does NOT collect, store, or transmit any user data.**

This extension:

- Does not collect personal information
- Does not track user behavior
- Does not use cookies
- Does not use analytics
- Does not transmit any data to external servers
- Does not store any data locally beyond what's required for basic functionality

## What the Extension Does

The extension performs a single, simple function:

1. Detects when you visit a GitHub repository page
2. Adds a "Zread" button to the repository navigation bar
3. When clicked, opens the corresponding Zread.ai documentation page in a new tab

All processing happens locally in your browser. No data ever leaves your device.

## Permissions Explained

The extension requires the following permissions:

### `host_permissions: ["https://github.com/*"]`

- **Purpose**: Required to inject the Zread button into GitHub repository pages
- **Scope**: Only activates on github.com domains
- **Data Access**: The extension only reads the current repository URL to construct the Zread.ai link. This information is not stored or transmitted.

## External Links

When you click the Zread button, you will be directed to `zread.ai`. This is an external website operated by a third party and has its own privacy policy. GitHub Zread Button is not affiliated with Zread.ai and is not responsible for their privacy practices.

## Third-Party Services

This extension does not use any third-party services, APIs, or analytics tools.
